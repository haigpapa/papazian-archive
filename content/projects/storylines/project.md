# STORYLINES
 
## Thesis
Narrative becomes spatial when moments, causes, characters, and themes are organized by relation rather than chapter order.
 
## Short Description
STORYLINES is a browser-based graph interface for exploring literary corpora as force-directed networks of moments and relationships.
 
## Full Description
Source texts are converted into readable nodes and explicit edges, with connection strength influencing proximity in the field. A visitor can move from an overview to a single moment, trace a causal chain, or inspect a thematic cluster without losing the larger structure. The project demonstrates how spatial navigation can reveal nonlinear narrative architecture, while its next documentation step is a concise recording that shows a real corpus and discovery path.
 
## Highlights
- Cinematic Rail Type: 8-card secondary proof rail
- Interaction: overview, node inspection, causal path, and thematic cluster
- Method: explicit graph relations rendered through force-directed layout
- Status: functioning web application; concise public walkthrough still needed
