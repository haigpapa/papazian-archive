# Kardia

## Thesis
A body-signal instrument that turns physiological state into sound without trapping the user in numbers.

## Short Description
Kardia is a mobile biofeedback instrument that translates heart-rate behavior into texture, density, and calm.

## Full Description
The system begins by matching the body's energy with sonic texture, then guides it toward calm as the body changes. It avoids raw BPM displays that create anxiety loops, abstracting biosignal into density, texture, and movement. The body becomes an instrument without being reduced to a dashboard.

## Highlights
- Biofeedback instrument based on the Iso-Principle
- Heart rate shapes harmonic density without displaying BPM
- Mobile-first interaction model for calm and attention
- Extends body-as-signal logic into therapeutic territory
