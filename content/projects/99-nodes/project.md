# 99 NODES
 
## Thesis
An archive can reveal how a practice is connected without reducing every work to the same level of importance.
 
## Short Description
99 Nodes is the spatial engine behind the Papazian Archive, organizing works, evidence, and relations as a navigable field.
 
## Full Description
The system gives the same body of work several deliberate forms: a curated Works spine, a relational Atlas, a searchable Index, and project-level cinematic rails. “99 Nodes” names the broader editorial constraint, while the public Works route foregrounds 20 projects. Nodes receive different weight according to authorship, evidence, relation, and consequence; archival fragments can support a parent project without competing with it as equal destinations.
 
## Highlights
- Cinematic Rail Type: 12-card meta-archive rail
- Public structure: 20-work spine within a larger relational archive
- Modes: Works, Atlas, Index, and project rail
- Status: active archive engine with continuing editorial and evidence review
