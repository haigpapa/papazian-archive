# 3D Beat Synth

## Thesis
A browser instrument that turns rhythm into an object you arrange in space.

## Short Description
3D Beat Synth is a Web Audio and Three.js instrument where beats become spatial forms rather than boxes in a grid.

## Full Description
Each beat cell is a physical object in 3D space, and composition becomes arrangement rather than linear sequencing. The work makes technical experimentation immediately legible: sound, interface, gesture, and spatial position become the same action.

## Highlights
- Browser-native instrument using Web Audio and Three.js
- Beat matrix arranged in 3D space instead of a flat grid
- Stereo field mirrors the spatial arrangement
- Rhythm treated as sculpture
