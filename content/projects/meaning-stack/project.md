# THE MEANING STACK
 
## Thesis
Culture changes how a system should interpret a signal; it cannot be safely added as metadata after the fact.
 
## Short Description
The Meaning Stack is a federated architecture for moving embodied signals through contextual processing, human audit, and spatial or sonic output.
 
## Full Description
The architecture begins with a sensorium—voice, movement, sound, text, or physiological input—and preserves provenance as the material is analyzed and organized. A conductor layer applies project-specific constraints; a verification layer separates observation from interpretation; a stage layer turns the result into something a person can hear, inspect, or move through. The stack is a shared technical framework rather than a single application, with components appearing across DERIVE, HAH-WAS, TEBR, and the archive.
