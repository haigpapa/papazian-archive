# Photon+

## Thesis
A browser twin of the Space Time Tuning Machine, translating a live tuning ritual into playable software.

## Short Description
Photon+ is a grid-based browser instrument that turns STTM's subtractive and spatial logic into an interactive web object.

## Full Description
Built with Web Audio and 3D interface logic, the work carries a performance system into the browser without reducing it to documentation. It is both a tool and a diagram of the original instrument: a way to rehearse sound, light, and memory through interaction.

## Highlights
- Digital twin of Space Time Tuning Machine
- Browser instrument built with Web Audio and 3D interaction
- Part of the Meaning Stack software family
- Turns subtractive composition into interface behavior
