# LOCALIZATION GAP
 
## Thesis
Generative music systems can imitate a regional style while erasing the tuning structures that make the music culturally specific.
 
## Short Description
The Localization Gap is an ongoing audit of generative-music outputs using structured prompting, reference listening, provenance records, and spectral analysis.
 
## Full Description
The research compares generated tracks with documented modal and microtonal references, combining computational signal analysis with culturally informed listening. Repeated errors are organized into a failure taxonomy so that model behavior can be compared across prompts, outputs, and time. The corpus includes hundreds of generated tracks, but every public result must retain model version, test date, source, method, and interpretive limits. HAH-WAS and MaqamAI translate the audit into evaluation and education tools.
 
## Highlights
- Cinematic Rail Type: 16-card paired rail
- Corpus: more than 400 generated-track records under active audit
- Method: structured prompts, reference listening, spectral comparison, and failure classification
- Status: ongoing research; public results require timestamped methods and limitations
