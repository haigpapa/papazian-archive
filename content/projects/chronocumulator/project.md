# CHRONOCUMULATOR
 
## Thesis
Musical time can accumulate through returning cycles instead of advancing along a single linear track.
 
## Short Description
Chronocumulator is a circular sequencing concept in which sound events are placed on rotating concentric rings and altered through recurrence.
 
## Full Description
Each ring carries events at its own cycle, allowing patterns to meet, separate, and return in changing combinations. Rather than treating a loop as exact repetition, the system is designed to retain residue and permit tuning changes between passes. Microtonal pitch relationships become visible alongside rhythm, connecting listening, geometry, and performance. The archive currently documents the interaction model, phased architecture, and interface studies; implementation status should remain explicit on every proof card.
 
## Highlights
- Cinematic Rail Type: 10-card lead system rail
- Interaction: place an event, rotate a ring, hear recurrence, alter the next pass
- Method: concentric timing cycles and microtonal pitch relationships
- Status: architecture and interface specification documented; playable proof remains in development
