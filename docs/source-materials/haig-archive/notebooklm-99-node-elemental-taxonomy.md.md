Six Domain Cards

#### The export proposes a six-card / six-slide taxonomy:

Sound: The Sensorium

Structural nodes: Space Time Tuning Machine, 3D-Beat-Synth.

Core copy direction: music is not ornament; it is a sensor for measuring the acoustic impedance of political spaces. The body becomes a tuning fork. Hacked performance hardware extracts modern myth from displacement noise.

Visual logic: black-ink technical blueprint of hacked violin, wireframe hand, digital grid.

Text: The Archive

Structural nodes: Sometimes I Wake Up Elsewhere, SparrowOS.

Core copy direction: bureaucratic surrealism documents exile through forms, audits, and avian classification systems. Memory becomes geography; the displaced subject is monitored by an administrative mythology.

Visual logic: weathered government form, monospace glitch text, avian classification stamp.

Image: The Visual Metaphor

Structural nodes: The Autopsy Interface, Nebucat.

Core copy direction: visual decay is the architecture of resolution loss. Glitch is not technical error; it is authentic transmission from a fractured identity.

Visual logic: datamoshing, pixel sorting, corrupted neural archive, rigid geometric frames.

Code: The Mechanics

Structural nodes: DERIVE, STORYLINES.

Core copy direction: code is the newest border drawn against the exiled body. Tuning rules preserve cultural friction and prevent algorithmic erasure.

Visual logic: force-directed constellation graph, vector-embedding flowchart, stark black/white over emerald.

Space: The Shelter

Structural nodes: photon+, MEKENA NYC.

Core copy direction: home is obsolete as a static concept. Space becomes an instrument engineered for survival, from adaptive reuse to optical sequencers.

Visual logic: isometric cross-sections, light beams, walls, human figures.

Systems: The Protocol

Structural nodes: The Meaning Stack, Systems Choreography.

Core copy direction: the work designs frameworks of possibility. Art, code, and policy operate as interchangeable building blocks for a sovereign creative operating system.

Visual logic: layer-cake matrix, Sensorium-to-Veracity Shield diagram, strict grid.
